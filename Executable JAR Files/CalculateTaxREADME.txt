This program should be run from command line.
In order to calculate tax enter a 4 digit employee number.
The program will then ask you for an employee salary, enter a number.
The program will then ask if you want to save this record, select y.
Press enter to return to the menu
Press 2 and enter the employee number you used above
The program will display the tax on the income.

You can change the tax rates and income brackets by manipulating the taxrates.txt file
